## Project Description

This project is part of the Open Source Software course. It involves analyzing an open-source software (VLC Media Player) and demonstrating Linux administration skills through shell scripting.

---

## Shell Scripts Overview

### Script 1 – System Identity Report

Displays system details such as user name, kernel version, uptime, and system information.

### Script 2 – FOSS Package Inspector

Checks if VLC is installed and displays version and description.

### Script 3 – Disk and Permission Auditor

Displays permissions, owner, and size of important directories.

### Script 4 – Log File Analyzer

Analyzes a log file and counts occurrences of a keyword.

### Script 5 – Open Source Manifesto Generator

Generates a personalized open-source philosophy statement.

## How to Run the Scripts


Repeat for other scripts.
